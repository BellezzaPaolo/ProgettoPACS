Detailed Description

Implementation of generic boundary-point selection.

The base Boundary_Condition provides:

  • filter(X): selects points on the boundary using the Python geometry mask (geom.on_boundary) and the user-provided predicate on_boundary.
  • collocation_points(X): currently equal to filter(X).
