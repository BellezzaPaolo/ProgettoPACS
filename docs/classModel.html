Detailed Description

template<class NetT>
class Model< NetT >

Minimal "Model" wrapper (in the spirit of DeepXDE's Model): it holds the data object, the neural network, and an optimizer/trainer.

Design choice: Model::train() is a thin wrapper that delegates to the real training loop implemented in Optimizer::train(...).

The PDE residual itself remains a callable (typically a lambda in main.cpp) stored inside Pde.

Plot training (and optionally test) loss history.

Delegates to the compiled optimizer's plot_cost_history(). The plot is produced via embedded Python/matplotlib (so it acquires the GIL).

Call this after train() if you want a visualization of the logged history.

Append a training run row to a CSV file (Python-compatible schema).

Mirrors the Python Model.save_data(...) helper. CSV schema: optimizer_name,batch_size,lr,final_budget,budget,n_fine,final_loss,epochs,time_train

The final_budget field follows your Python convention: the remaining budget after training ends (budget - budget_used).
