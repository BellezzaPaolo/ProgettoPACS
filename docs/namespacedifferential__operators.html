Compute the Laplacian of a scalar field $u(x)$ using torch autograd.

For each sample $x \in \mathbb{R}^{\mathrm{dim}}$, this returns $$\Delta u(x) = \sum_{i=1}^{\mathrm{dim}} \frac{\partial^2 u}{\partial x_i^2}.$$

Expected tensor shapes:

  • x: [N, dim] with requires_grad=true
  • u: [N, 1] or [N] (or [N, out_dim] with a selected component)
Parameters
uField values evaluated at x.
xInput coordinates. Must have requires_grad=true.
componentComponent index to use when u is [N, out_dim].
Returns
Tensor of shape [N, 1] containing the Laplacian values.

First derivatives: grad_u = du/dx, shape [N, dim].

Second derivatives: $\sum_i \partial^2 u/\partial x_i^2$ (trace of Hessian).
