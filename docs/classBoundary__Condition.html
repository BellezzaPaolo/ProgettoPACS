Detailed Description

Abstract base class for boundary conditions.

This is the parent class for all boundary conditions (Dirichlet, Neumann, Robin, Periodic, etc.).

◆ filter()

tensor Boundary_Condition::filter ( const tensor &  X) const
virtual

Filter points that are on the boundary.

This method calls the Python geometry predicate geom.on_boundary(X) to get a boolean mask and then applies the user-provided on_boundary(x_i, mask_i) callback to decide which points to keep.

Parameters
XInput points matrix [N, dim].
Returns
Filtered points of shape [M, dim] with M <= N.

X is assumed to be of shape [N, dim].

Convert X to a NumPy array to call Python geom.on_boundary (DeepXDE-style API).

geom.on_boundary returns a boolean array of shape [N].

Build a 1D tensor view for the i-th point, shape [dim].

Stack the selected rows into the filtered tensor.

Callback deciding whether a point lies on the boundary.

The callback signature is on_boundary(x, on_bc_mask) where:

  • x is a 1D tensor view of one point, shape [dim]
  • on_bc_mask is the boolean mask element returned by Python geom.on_boundary(X).
