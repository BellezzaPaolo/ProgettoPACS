Detailed Description

template<class NetT>
class ParaflowS< NetT >

ParaFlowS optimizer/trainer for the torch PINN pipeline.

This implements the ParaFlowS scheme:

  • A coarse operator computes a large-step candidate (stored internally).
  • A fine operator performs n_fine small GD steps (updates parameters).
  • A correction loop runs up to n_coarse coarse steps while enforcing monotonic decrease w.r.t. the fine loss.

The interface matches the other project optimizers: it derives from Optimizer<NetT> and implements train().
