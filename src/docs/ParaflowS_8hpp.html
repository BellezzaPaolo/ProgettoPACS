Detailed Description

ParaFlowS optimizer/trainer (coarse/fine + correction) for PINNs.

ParaFlowS alternates between:

  • a coarse solver that builds a candidate parameter state (stored in bff1)
  • a fine solver that performs n_fine standard SGD steps on the live network
  • a correction loop that combines coarse candidate + correction and enforces monotonicity with respect to the fine loss.

For PINNs, input batches are created as leaf tensors with requires_grad=true to enable autograd-based differential operators.
